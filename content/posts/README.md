# Posts

This folder contains **Posts** - finished pieces of writing meant for sharing or publishing. Posts are typically:

- More structured and polished
- Centered around a specific topic
- Meant for public consumption
- Longer and more comprehensive
- Include proper citations and references

## Structure

Posts should have proper frontmatter with:
- `title`: Clear, descriptive title
- `tags`: Relevant tags for categorization
- `date`: Publication date
- `description`: Brief summary for SEO
- `draft`: Set to `false` for published posts

## Examples

- Tutorial guides
- Technical deep-dives
- Project showcases
- Research summaries
- Opinion pieces
