# Notes

This folder contains **Notes** - rough ideas, drafts, and learning materials. Notes are typically:

- Short and unpolished
- Quick thoughts and ideas
- Learning materials and references
- Work-in-progress content
- Personal knowledge base entries

## Structure

Notes should have frontmatter with:
- `title`: Descriptive title
- `tags`: Relevant tags for categorization
- `date`: Creation date
- `draft`: Can be `true` for incomplete notes

## Examples

- Quick references
- Learning notes
- Code snippets
- Concept explanations
- Personal thoughts
- Research notes
